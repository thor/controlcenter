    Handover¶

    Handover is another project maintained by VATSIM Scandinavia. It is an alternative to VATSIM Connect which provides you with the ability to

    • GDPR Complicance: Centralise user authentication in one location to easily scrub systems of user data.
    • Authorization: Use Handover to authorize or reject user requests to other websites.
    • OAuth2 compatible: Integrate Handover instead of VATSIM Connect directly with other services for your division.

    See GitHub for more

    Find out more about VATSCA's Handover project by visiting its repository.
